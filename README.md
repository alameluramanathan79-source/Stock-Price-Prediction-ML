# Stock Price Prediction Using Machine Learning

## Overview

This project predicts stock prices using Machine Learning techniques in Python. It includes data preprocessing, visualization, feature engineering, model training, and prediction.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Features

- Data preprocessing
- Data visualization
- Feature engineering
- Model training
- Stock price prediction
- Performance evaluation
